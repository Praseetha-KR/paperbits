PaperBits

Creative people's scrap board
To keep bits of paper on the web!

Login/Register

Features

PaperBits provides a solution to save the bits of inspirations on the web for the creative people. A board where you can store colors, images, notes, reference urls and also sticky notes for noting down important tasks.

Colors

  • Store random values from color picker extension/add-ons
  • Automatic detection and color space conversion (hex, rgb, hsl)
  • Look for combinations from saved colors

Images

  • Moodboard with random inspirational images
  • Tags to categorize images (to be implemented)

Links

  • Links to references or read-it-later articles
  • Better than bookmarks as it provides website preview
  • Delete easily after reading
  • Tagged categories (to be implemented)

Notes

  • Quick notes for code help/conference notes/reference snippets etc.
  • Easy edit and save
  • Categorize as notebooks (to be implemented)

Todo

  • Todo task lists as simple as a sticky note!
  • Task lists are colored based on priority
  • Sort tasks based on priority & date.
